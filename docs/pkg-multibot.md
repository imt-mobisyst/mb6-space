# Package MultiBot

The goal of this package is to allow several robots to work together.

## Get Started

It is possible to use simple clone and install script : 

```sh
./bin/clone-multibot
source ./bin/run-commands.bash
```

This package has its own documentation space >> [imt-mobisyst.github.io/pkg-multibot](https://imt-mobisyst.github.io/pkg-multibot)
